# Video Speed Enhancer Browser Extension

This extension helps you easily increase the playback speed of videos. The speed adjustment is optimized with flexible settings and a user-friendly interface.

## Features

- Increase and reset video playback speed.
- Supported sites: YouTube, Vimeo, and other HTML5 video platforms.
- Easy switching between speed options.

## Installation

1. Open your browser's extension store (Chrome Web Store, Firefox Add-ons, etc.).
2. Search for "Video Speed Enhancer" extension.
3. Download and follow the installation instructions provided.

## Usage

1. Open a video on a supported site.
2. Click the extension icon in your browser toolbar.
3. Select the desired video speed; the change will be applied immediately.

## Settings

- The default speed range can be customized according to your preferences.
- Access additional speed options in the advanced settings menu.
